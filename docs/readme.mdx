# Neumorphic UI Docs

`neumorphic-ui` is a library of components based on the concept of 
neumorphism.

## Installation

```
npm install neumorphic-ui
```

## Usage

Generally, components in `neumorphic-ui` can be imported as any React component

```js
import { ExampleComponent } from 'neumorphic-ui';
```

[See the official docs](https://github.com/dev-saeed/neumorphic-ui)
